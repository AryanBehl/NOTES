# 📝 Notes App – Simple & Fast Note Manager

A clean and lightweight **Notes App** built using **HTML, CSS, and JavaScript**.
This application allows users to create, search, and delete notes with data saved in the browser using **Local Storage**.

---

## ✨ Features

* ➕ Add new notes instantly
* 🔍 Search notes in real-time
* ❌ Delete notes easily
* 💾 Data stored using Local Storage
* ⚡ Fast and responsive UI
* 🧠 Beginner-friendly project

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage API
